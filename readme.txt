== Description ==

Prices by User Role Lite it is a plugin designed to extend the default WooCommerce functionality by hiding Add to Cart button and product prices from the user roles. 

**Free version features**

*  Hide Add to Cart Button for User Roles 
*  Hide Prices for User Roles 

**For more features and amazing support get Prices by User Role version:**

*  Prices by User Role for Simple Products
*  Prices by User Role for Variable Products
*  Hide Add to Cart Button for User Roles 
*  Hide Prices for User Roles
*  Text for Registered Users with Hidden Price
*  Pricing Roles
*  Discount for the user role group
*  Disable discount per product
*  Markup for user role group
*  Hide the “Add to Cart” button from non registered users
*  Hide the prices from non registered users
*  Text instead of the price for the non registered user
*  Hide the “Add to Cart” button from the specific user role group
*  Hide the prices from the specific user role group
*  Text instead of the price for the specific user role group
*  Display the savings percentage
*  Custom user groups
*  WooCommerce CSV import for simple and variable products
*  Set different prices for different roles for individual products
*  Charge different prices for each variation in a variable products
*  Hide the price for non registered users for individual product
*  Hide the price and the “Add to Cart” button from user groups for individual products
*  Support WooCommerce REST API
*  WPML translation ready

== Installation ==

You can install Prices by User Role Lite via the WordPress plugin uploader without unzipping the file:
<ol>
<li>Log into your WordPress admin panel</li>
<li>Navigate to Plugins > Add New</li>
<li>Click Upload</li>
<li>Click Choose File and select the woocommerce-prices-by-user-role-lite.zip</li>
<li>Click Install Now</li>
<li>Activate the plugin through the Plugins menu in WordPress</li>
<li>Go to new WooCommerce section in Admin Bar</li>
</ol>
